## Step-by-step to run the project

1. Make sure you have Node.js and npm installed on your system.
   You can check if they are installed by running the commands node -v and npm -v in your terminal.
   If they are not installed, you can download and install them from
   [the official Node.js website](https://nodejs.org/en/download/current).
2. Install MongoDB on your system. [MongoDB](https://www.mongodb.com/docs/manual/installation/)
3. Make sure you already enable MongoDB server. The way can be different with different OS
4. Open the project -> open terminal -> typing `npm install`
5. Enter `npm start` to start the server in the terminal
6. Access to URL showed in the terminal
